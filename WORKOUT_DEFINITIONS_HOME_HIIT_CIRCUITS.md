## WORKOUT 1: 10 MIN QUICK BURN
**Focus:** Maximum results in minimum time (most popular format)

### BEGINNER (4 exercises - 30s work / 30s rest, 2 rounds)
**Philosophy:** Learn HIIT basics with manageable intensity
1. **Marching In Place** (30s work / 30s rest) - Cardio warm-up
2. **Squat Reach** (30s work / 30s rest) - Lower body (squat without jump)
3. **Incline Push-Up** (30s work / 30s rest) - Upper body (hands elevated)
4. **Standing Oblique Crunch** (30s work / 30s rest) - Core engagement

**Repeat 2 rounds = 8 minutes + warm-up/cool-down = 10 min**

**Why this works:** Kaiser Permanente 10-min HIIT uses this exact structure. Beginners need 1:1 work:rest ratio. Low-impact versions (no jumping) prevent injury. This teaches interval training basics.
